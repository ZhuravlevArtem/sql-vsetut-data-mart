# SQL-проект: витрина данных маркетплейса «ВсёТут»

Учебный SQL-проект по разработке витрины данных и решению ad-hoc задач на основе базы данных маркетплейса «ВсёТут».  
Проект выполнен в рамках обучения на курсе Data Science.

## Описание проекта

Цель проекта — подготовить витрину данных `product_user_features` с агрегированными характеристиками поведения пользователей и на её основе решить несколько аналитических задач.

В проекте используются данные о заказах, пользователях, оплатах, товарах и отзывах.  
Основной результат — витрина с пользовательскими метриками, которую можно использовать для последующего анализа клиентского поведения.

## Что реализовано в проекте

### 1. Подготовка витрины данных

Сформирована витрина `product_user_features`, содержащая агрегированные характеристики пользователей:

- регион пользователя;
- дата первого и последнего заказа;
- длительность активности пользователя;
- общее количество заказов;
- средний рейтинг заказов;
- количество и доля отменённых заказов;
- суммарная и средняя стоимость заказов;
- количество заказов в рассрочку;
- количество заказов с промокодами;
- признаки использования денежных переводов, рассрочки и отмен.

При расчётах использовались только заказы со статусами `Доставлено` и `Отменено`, а также топ-3 региона по количеству заказов.

### 2. Решение ad-hoc задач

На основе витрины были решены 4 аналитические задачи:

#### Сегментация пользователей
Пользователи разделены на сегменты по количеству заказов:
- 1 заказ;
- 2–5 заказов;
- 6–10 заказов;
- 11 и более заказов.

Для каждого сегмента рассчитаны:
- количество пользователей;
- среднее количество заказов;
- средняя стоимость заказа;
- общее количество заказов.

#### Ранжирование пользователей
Выведены пользователи с 3 и более заказами, отсортированные по среднему чеку.  
Дополнительно рассчитаны:
- ранг пользователя по среднему чеку;
- доля пользователя в общей стоимости заказов.

#### Статистика по регионам
Для каждого региона рассчитаны:
- общее количество клиентов;
- общее количество заказов;
- средняя стоимость заказа;
- доля заказов в рассрочку;
- доля заказов с промокодами;
- доля пользователей, совершивших отмену заказа хотя бы один раз.

#### Активность пользователей по первому месяцу заказа
Пользователи сгруппированы по месяцу первого заказа в 2023 году.  
Для каждой группы рассчитаны:
- количество клиентов;
- число заказов;
- средняя стоимость заказа;
- средний рейтинг;
- доля пользователей, использующих денежные переводы;
- средняя продолжительность активности пользователя.

Дополнительно рассчитаны:
- ранг месяца по количеству новых клиентов;
- накопленное количество клиентов по месяцам.
